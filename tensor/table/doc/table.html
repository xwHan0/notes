Introduce

Table(表，缩写T)是一种二维有限离散数据结构。

Table的第一个维度sequence，第二个维度为map。可以使用嵌套一维数组表示为：

    tblName = [{field0: value0, field1: value1, ..., fieldN: valueN}]

Query

Query使用一个或者多个table和其它一些参数，计算出一个新的table。输出table使用field来标识，计算使用输入table的field来标识。

Syntax

SELECT field-or-fieldProc-list
FROM table-or-tableFunc-list
[WHERE fieldPred-or-fieldProcPred-list]
[GROUP BY fieldPred-or-fieldProc-list
  [HAVING fieldPred-or-fieldProcPred-list]
]
[ORDER BY fieldPred-or-fieldProcPred-list]
