Vector

向量是由有限个元素组成的有序集合,表示为:a=[a1,a2,...,aN]。其中a为向量，a1,a2,aN为向量a的元素。 元素在向量中的位置称为下标。

Geometry View

N维向量a=[a1,a2,...,aN]表示在N为线性空间中，过坐标原点和点(a1,a2,...,aN)的有向线段。

Matrix

Geometry View

把M*N的矩阵A按列分解为多个列向量的集合: A=[c[1],c[2],...,c[N]],n为 列向量c[1],c[2],...,c[N]的无关度. 则矩阵A可以表示以下几种几何含义(按行向量分解原理一样)：

  • 空间体意义：M维线性坐标系中，由列向量c[1],c[2],...,c[N]作为边而围成的n(n<=N)维平行多面体
    • n<=M: 平行多面体维度不会超过矩阵所在线性空间的维度
    • n<=N: 当c[i]存在线性相关向量时，n会减小
    • 当n<N时，称该平行多面体为伪N维平行多面体
      • 该平行多面体有N个自由基，但仅有n个维度
      • 该平行多面体由最"外侧"的n个向量张起
    • n为矩阵A的秩
      • n=rank(A)
  • 空间意义：M维线性坐标系中，由向量c[1],c[2],...,c[N]作为单位基而张成的N维新线性空间坐标
    • 同样：n=rank(A)<=min(M,N)
    • 当n<N时，称该新线性空间为伪N维线性空间
      • 该线性空间有N个自由基，但仅有n个维度
      • 每个维度坐标值是由N个自由基线性求和贡献的

例如：

矩阵A=[[3,1],[2,2]]可表示以下几何意义

  • 2维空间中以向量[3,2]和[1,2]组成的2维平行四边形
  • 或者，2维空间中以向量[3,2]和[1,2]为坐标基组成的新的2维平面坐标系统

矩阵A=[[3,1],[1,2],[1,1]]表示以下几何意义

  • 3维空间中以向量[3,1,1]和[1,2,1]组成的2维平行四边形
  • 或者，3维空间中以向量[3,1,1]和[1,2,1]为坐标基组成的新的2维平面坐标系统

矩阵A=[[3,2,1],[1,4,1]]表示以下几何意义

  • 2维空间中以向量[3,1], [2,4]和[1,1]组成的四边形
  • 或者，2为空间中以向量[3,1], [2,4]和[1,1]为基组成的2为坐标系
    • 该2维坐标系有3个坐标维度(相互有重合)

矩阵A=[[3],[2]]表示以下几何意义

  • 2维空间中以向量[3,2]描述的1维线段
  • 或者，2维空间中以向量[3,2]为坐标基组成的新的1维线坐标系统

矩阵A=[[3,1]]表示以下几何意义

  • 1维空间中以向量[3]和[1]描述的1维线段(本质上为方向重合的2个向量)
  • 或者，1为空间中以向量[3]和[1]为坐标基(向量方向重合)的新的重合的2为线坐标系统
    • 该线性坐标系需要2个坐标维度
    • 2个坐标维度按照3:1的权重加权求和后得到最终的坐标位置

对于矩阵A=[[3]]，因为只有一个维度，所以只能表示一维坐标轴的一个有向线段，或者以3为坐标基的新1维坐标系。

矩阵也可以按行分解:A=trans([r1,r2,...,rM])，得出与按列分解类似的结论。
