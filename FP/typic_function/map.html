Syntax

map(proc, iterator0, iterator1, ...)

parameter

  • proc: process action function(s)
  • iterator: process data set

return

Feature

Default Process

Uses 'identifier' function to process when proc parameter is ignore.

Process Set

When proc parameter is not a function but a list, iterates each proc in this list and return

Collection Set

Supports one or more collections.

Minimum Iterator Fetch

Iteratetion stops after any iterator is ended.

Minimum Parameter Fetch

Assume the number of parameters is P and the number of iterator is I,

  • P >= I: fetch the first P iterators to process.
  • P < I: return a closure who needs more (P-I) parameters.

Delay Execute

Return a closure who needs iterator(s) when iterator(s) parametera ignore.

Return Type Comprehence

Define: The collection type of return back is same with the first iterator parameter.
