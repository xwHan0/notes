多表联合查找。

Syntax

SELECT ... FROM tblA
JOIN-TYP tblB ON join-predA
JOIN-TYP tblC ON join-predB
...
JOIN-TYP tblZ ON join-predZ
other-select-statement

处理过程为：

  1. 枚举tblA中的表项a
  2. 按照join-predA条件在tblB中寻找匹配的项
  3. 若找到，则进行JOIN成功操作
  4. 若没有找到，则进行JOIN失败操作
  5. 枚举下一个a

      6. JOIN类型

      INNER JOIN

      OUTER JOIN LEFT

      OUTER JOIN RIGHT

      CORSS JOIN
