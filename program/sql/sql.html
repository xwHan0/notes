Tutorial

多表联合查找。

Syntax

SELECT ... FROM tblA
JOIN-TYP tblB ON join-predA
JOIN-TYP tblC ON join-predB
...
JOIN-TYP tblZ ON join-predZ
other-select-statement

Work Flow

  1. 按GROUP BY字段进行联合分类
  2. 生成一个临时表,其内容为:
    • 每个分类域段作为一列
    • 满足该分类的所有表项"合并"成一个域段
    • 若存在ALL关键字，不满足分类条件的项单独作为一个聚合项
  3. 使用HAVING条件过滤临时表内容(HAVING操作临时表)

聚合函数

  • sum(column): 求和
  • max(column): 最大值
  • min(column): 最小值
  • avg(column): 平均值
  • count(column): 统计记录数

Constraint

  • SELECT选择的域只能为以下2种：(因为SELECT操作的是GROUP产生的临时表)
    • GROUP BY的分类域
    • 聚合函数
