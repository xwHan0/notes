Introduce

Content

一个JSON由以下可选部分组成。

Number

数字值

Syntax
    number ::= [0-9]+(\.[0-9]+)?

String

字符串

Syntax
    string ::= [.]+

根目录

Syntax
    root ::= [\/]

Linux下为'/'，Windows下为'\'。缺省表示相对当前目录的相对路径(见direction小节)。

direction

目录

Syntax
    direction ::= ([\w .]+[/]?)+

特别的目录标识

  • '.': 当前目录
  • '..': 父目录
  • '...': 祖目录

file

文件

Syntax
    file ::= [\w .]+

ext

文件扩展

Syntax
    ext ::= [\w]+

query

参数

Syntax
query ::= ?query-form&query-form...
query-form ::= query-key[=query-jsonlike-value]
  • query-jsonlink-value: JSON格式的查询值。若忽略，则query-key为Boolean变量，且其值为True
