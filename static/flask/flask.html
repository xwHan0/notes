Usage

Config

配置class。

提供如下的基本配置方法

  • 函数配置：直接解析key:val对
    • Syntax: config.from_mapping(key0=val0,...,keyN=valN)
  • 文件配置：读取文件配置项
    • Syntax: config.from_file(fileName, slience=False)
    • Syntax: config.from_object()
  • 直接访问
    • Syntax：config[strKey]=val

Request

HTTP请求对象

  • method:请求的HTTP方法。取值为：
    • "GET":
    • "POST":
    • "PUT":
    • "DELETE":

Session

  • Access (Read and Write)
    • [key]: Access element
    • get(key)
  • clear(): 清空

Redirect

Syntax: strResp redirect(url)
